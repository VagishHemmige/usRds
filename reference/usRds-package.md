# usRds: A Package to Automate Processing and Access to the USRDS

Provides tools to simplify access to and processing of data from the
United States Renal Data System (USRDS). Supports multiple raw formats
(SAS, CSV, Parquet), automated file detection, optional value and
variable labeling using USRDS appendices, and functions to filter claims
data by diagnosis and procedure codes. Designed for reproducible,
large-scale cohort construction and longitudinal analysis.
